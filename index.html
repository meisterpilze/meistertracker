How to connect

Thunderbird / GNOME Calendar: Add a new "CalDAV" or "network calendar" and paste the CalDAV URL above.

Android (DAVx5): Install DAVx5 from F-Droid or Play Store. Add account with "Login with URL" and paste the URL.

Apple Calendar (iPhone/Mac): Settings → Calendar → Accounts → Add CalDAV account. Server = the URL above.

How it works

Base calendar: All tasks go to a shared meisterpilze-tasks calendar.

Per-person calendars: When enabled, tasks assigned to a team member get their own calendar (e.g. meisterpilze-jonas). Unassigned / company-wide tasks stay in the shared calendar.

Two-way: Tasks are written to calendar files on this server. Calendar clients can read them via CalDAV. Changes made in calendar apps (marking done, editing) are saved back to the .ics files.

Calendars on disk: Stored in /calendars/ folder next to data.json. Each calendar is a folder with .ics files inside.
