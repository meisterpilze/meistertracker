How to connect

Thunderbird / GNOME Calendar: Add a new "CalDAV" or "network calendar" and paste the CalDAV URL above.

Android (DAVx5): Install DAVx5 from F-Droid or Play Store. Add account with "Login with URL" and paste the URL.

Apple Calendar (iPhone/Mac): Settings → Calendar → Accounts → Add CalDAV account. Server = the URL above.

So funktioniert es

Betriebskalender (meisterpilze): Alle Aufgaben, Termine und Batch-Fälligkeiten landen hier. Jeder im Betrieb sieht diesen Kalender.

Persönliche Kalender (benutzername): Jeder User-Account bekommt automatisch einen eigenen Kalender. Zugewiesene Aufgaben erscheinen dort zusätzlich.

Sichtbarkeit: Standardmäßig sind alle Aufgaben für alle sichtbar (im Betriebskalender). Einzelne Aufgaben können als "Nur für zugewiesene Person" markiert werden — diese erscheinen dann nur im persönlichen Kalender.

Zwei-Wege-Sync: Aufgaben werden als Kalender-Dateien auf dem Server gespeichert. Kalender-Apps lesen sie via CalDAV. Änderungen in der Kalender-App werden zurückgespeichert.
